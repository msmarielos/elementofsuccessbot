# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Создание бота в Telegram

1. Откройте Telegram
2. Найдите [@BotFather](https://t.me/BotFather)
3. Отправьте `/newbot`
4. Следуйте инструкциям и скопируйте токен

## Шаг 3: Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_от_BotFather
BOT_USERNAME=your_bot_username

PUBLIC_BASE_URL=https://your-domain.com

TBANK_TERMINAL_KEY=ваш_terminal_key
TBANK_PASSWORD=ваш_password
```

## Шаг 4: Настройка планов подписки

Откройте `src/types/subscription.ts` и настройте планы подписки под ваши нужды.

## Шаг 5: Запуск бота

### Режим разработки:
```bash
npm run dev
```

### Продакшен:
```bash
npm run build
npm start
```

## Шаг 6: Тестирование

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Попробуйте команды:
   - `/plans` - посмотреть планы
   - `/buy` - купить подписку
   - `/my_subscription` - проверить подписку

## Интеграция платежной системы

Интеграция уже настроена под T‑Bank (WebView / `PaymentURL` + проверка через `GetState`).

Инструкция по настройке: `TBANK_SETUP.md`.
